Password Generator

サービスごとの記号ポリシーに対応したパスワード生成ツール

  • Web Crypto API で乱数生成（Math.random 不使用）
  • 完全クライアントサイド完結 — パスワードを外部サーバーへ送信しません
  • パスワードを端末に保存しません
  • カスタムプリセット（記号セットのみ）はブラウザの localStorage に保存されます
